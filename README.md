# LLM Evaluations

Static benchmarks vs production evals, LLM-as-judge with its biases, the eval-framework landscape (Inspect AI, Braintrust, Phoenix, Langfuse), drift detection and red-teaming.

**Live index:** https://brendanjameslynskey.github.io/LLM_Hub_Evaluations/

## Presentations in this series

| 02 | [LLM-as-Judge](https://brendanjameslynskey.github.io/LLM_Eval_02_LLM_as_Judge/) | live | Pairwise vs scoring; rubric, exemplars, CoT prompts; biases (positional, verbosity, self-preference, sycophancy); mitigation; constitutional eval; calibration; cost vs quality. |
